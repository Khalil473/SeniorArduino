#### Modules used:
- Arduino Nano
- DHT22
  - Data pin to D4
- SD Card
  - CS pin to D10
  - MOSI pin to D11
  - MISO pin to D12
  - CLK pin to D13
- DS1302 RTC
  - RST pin to D9
  - DAT pin to D7
  - CLK pin to D5
- HX711 ADC
  - DOUT pin to D3
  - SCK pin to D6
- HM-10 BLE
  - TX pin to D2
  - RX pin to D8
- Vibration Motor
  - VCC pin to A0
