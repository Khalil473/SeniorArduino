# DHT22 data Logger to SD

### A program that logges DHT22 data to SD each two seconds with, with appropriate files creating and error handling.

#### Modules used:

- DHT22
  - Data pin to D4
- SD card
  - CS pin to D10
  - MOSI pin to D11
  - CLK pin to D13
  - MISO pin to D12
- DS1302 RTC
  - RST pin to D9
  - DAT pin to D7
  - CLK pin to D5
